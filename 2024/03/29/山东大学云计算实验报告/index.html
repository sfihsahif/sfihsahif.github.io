山东大学云计算实验报告

学号：202100130079 姓名：赵文心 班级：计科21.6
实验题目：利用主流云平台搭建个人博客或网站

实验学时：2

实验日期：2024/3/29

实验目的：熟悉使用主流云平台并搭建个人博客或者网站。
具体包括：基于主流云平台，设计、实现个人博客或者网站的搭建，撰写实验报告（附带网站链接并可以访问），并在网站上呈现此次实验报告。

硬件环境：
联网的计算机一台

软件环境：
Windows

实验步骤与内容：
1、环境准备
（1）安装git
（2）安装node.js
（3）安装Hexo
利用命令：npm install hexo -g
（4）安装hexo依赖
利用命令：npm install –save hexo-deployer-git

2、git配置SSH key
（1）配置ssh
先生成ssh key，利用命令ssh-keygen -t rsa -C “897126654@qq.com“
可以看到在用户文件夹下生成了一个ssh文件夹，进入到ssh文件夹，复制id_rsa.pub文件里全部内容，接着打开github主页，设置好SSH key

（2）配置账号和密码
利用如下命令完成
$ git config –global user.name “sfihsahif”
$ git config –global user.email “897126654@qq.com“

3、搭建个人博客
（1）新建博客
新建一个保存博客的存放目录，进入到本地博客存放目录，先用命令 hexo init 初始化个人博客
（2）部署到github
新建一个github仓库，命名：sfihsahif.http://github.io
修改_config.yml
deploy:
type: git
repository: git@github.com:sfihsahif/sfihsahif.github.io.git
branch: master
用命令 hexo d 发布到github

4、编写博客
利用命令：hexo new ‘山东大学云计算实验报告’创建相应的文件夹，md文件去写博客。编写好之后，hexo g hexo d部署到github

结论分析与体会：
搭建个人博客是一个非常有意义的事情，可以帮助我记录学习过程和感想。在搭建个人博客的过程中，我学会了使用Hexo生成静态网页并进行部署，同时也了解了一些markdown的语法。完成这次实验的过程中也遇到了很多意料之外的错误，但都已经解决，提高了我的实践能力。
